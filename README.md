## **官网注册与使用步骤**

1. **访问官网**：[https://chat.openai.com](https://chat.openai.com)（需要翻墙）。
2. **注册账户**：
   - 确保有海外手机号可用，以便获取验证码。
   - 填写电子邮箱和设置密码完成注册。
3. **选择版本**：登录后选择 GPT-4 或 GPT-3.5。
4. **开始对话**：输入你的问题，体验原汁原味的 OpenAI 服务。
